# Third-Party Notices

This file tracks third-party components, dependencies, tools, and notices related to Bravo Fires FDC public releases.

## Current C# / Avalonia release line

Bravo Fires FDC v4.x is built with the .NET / Avalonia desktop stack. Before a stable release, confirm exact package versions from the private source repository lockfile/project files and update this page with any required license text or links.

Components to review for the v4.x C# release line:

- Microsoft .NET SDK / .NET runtime
- Avalonia UI
- Avalonia desktop runtime packages
- SkiaSharp / native rendering components
- HarfBuzzSharp / native text-shaping components
- System.Text.Json and other .NET framework libraries
