# Bravo Fires FDC Public Release Channel

This repository is the public release and update channel for **Bravo Fires FDC**, an Arma Reforger companion application.

The source code for Bravo Fires FDC is not stored in this repository. This repository is intended to host only public-facing release material such as:

- update metadata
- changelog entries
- release notes
- downloadable release packages
- legal notices
- third-party notices

## Update manifest

The desktop application should check the public update manifest from this repository:

```text
https://raw.githubusercontent.com/ProtocolGoose/BravoFiresFDC-Public/main/update_manifest.json
```

## Current release status

The current public manifest is prepared for the v3.0.x update-checking foundation. Release assets should be attached through GitHub Releases.

## Important notice

Bravo Fires FDC is a standalone companion tool for Arma Reforger gameplay. It does not connect to Arma Reforger, inject into the game, read game memory, or control the game.

Bravo Fires FDC is an independent project and is not affiliated with, endorsed by, or sponsored by Bohemia Interactive or any real-world military organization.

The source code is proprietary and is not distributed through this public repository.
