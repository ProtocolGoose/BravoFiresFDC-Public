## Application behavior

Bravo Fires FDC requires one-time EULA acceptance before the main application opens. If the EULA is declined, the application closes.

Acceptance is stored locally for the current Windows user so users are not prompted every time the app launches. If the legal agreement identifier or EULA URL changes in a future release, the app can require acceptance again.
