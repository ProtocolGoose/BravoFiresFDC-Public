# Bravo Fires FDC Privacy Policy

**Working public repository copy. Replace with your approved privacy policy before public release if your final wording differs.**

Bravo Fires FDC is designed as a standalone desktop companion application.

## Local data

The application may store local files such as layout preferences, notes, logs, changelog files, and local update metadata on the user's device.

## Update checks

When the user checks for updates, or when the app performs a silent startup update check, the app may request the public update manifest from this repository. GitHub may process standard web request data according to GitHub's own policies.

## No game connection

Bravo Fires FDC does not connect to Arma Reforger, inject into the game, read game memory, or control the game.

## No account system

The public desktop application does not require a user account for the local companion calculator features.

## Contact

Privacy-related questions should be directed to the project owner through the support/contact method provided with the official release.
