# Bravo Fires FDC v4.1.3-beta Release Notes

## Summary

v4.1.3-beta fixes 8-digit grid/keypad handling in the C# / Avalonia branch. 8-digit grids are now treated as already-refined positions, KP is ignored for 8-digit grids, and KP remains active for 6-digit grids only. This release preserves the v4.1 floating-module workspace and v4.1.2 update checker polish.

## User-facing changes

- Fixes 8-digit grid handling so KP is ignored when a grid already includes keypad-level precision.
- Disables KP fields in the Mission Inputs UI for 8-digit gun and target grids.
- Keeps 6-digit grid + KP behavior intact for coarse grid workflows.
- Preserves the v4.1 true floating-module workspace.
- Preserves the v4.1.2 update checker data-panel polish.
- Continues to download and verify update ZIP packages without automatic installation.

## Update behavior

This release continues the C# / Avalonia update workflow. The app checks `releases/latest.json`, compares versions, displays release details, downloads the release ZIP, and verifies SHA256 when a real hash is provided.

Automatic replacement/installation is not included yet.
